# User Flows

## Restaurant Owner Flow

### Registration

```
1. Visit nexaros.com
2. Click "Get Started"
3. Fill registration form:
   - Business type (10 options)
   - Restaurant name
   - Email
   - Phone
   - Address
   - City
   - State (Indian states)
   - Password
4. Verify email (planned)
5. Login to dashboard
6. Complete onboarding:
   - Add menu items
   - Configure tables
   - Add staff
7. Start using POS
```

### Subscription

```
1. Trial expires or needs upgrade
2. View pricing page
3. Select plan
4. Enter payment details
5. Confirm subscription
6. Features unlocked
```

## Staff Flow

### Clock In

```
1. Open app
2. Tap "Clock In"
3. Enter PIN
4. Confirm clock in
5. Start shift
```

### Take Order

```
1. Open POS
2. Select table or takeout
3. Add items to cart
4. Apply discounts
5. Confirm order
6. Print KOT
7. Order sent to kitchen
```

### Process Payment

```
1. View completed order
2. Tap "Pay"
3. Select payment method
4. Enter amount
5. Process payment
6. Print receipt
7. Table status → FREE
```

## Kitchen Flow

### Receive Order

```
1. Order received notification
2. View order details
3. Start preparation
4. Update status → PREPARING
5. Complete preparation
6. Update status → READY
7. Notify POS
```

### Handle Multiple Orders

```
1. View order queue
2. Sort by priority
3. Process orders
4. Update statuses
5. Mark as ready
```

## Manager Flow

### Daily Operations

```
1. Login to dashboard
2. View today's stats
3. Check orders
4. Monitor inventory
5. Review staff attendance
6. Generate reports
```

### Staff Management

```
1. View staff list
2. Create shifts
3. Assign staff
4. Review attendance
5. Handle requests
```

## Admin Flow

### Platform Management

```
1. Login to admin portal
2. View platform stats
3. Manage tenants
4. Handle subscriptions
5. Review support tickets
6. Configure settings
```

### Tenant Management

```
1. View tenant list
2. Select tenant
3. View details
4. Suspend/activate
5. View subscription
6. Handle issues
```

## Customer Flow

### Online Ordering

```
1. Scan QR code or visit link
2. View restaurant menu
3. Browse categories
4. Add items to cart
5. Enter table number
6. Place order
7. Track order status
8. Receive notification
```

### Reservation

```
1. Visit restaurant page
2. Check availability
3. Select date/time
4. Enter party size
5. Confirm reservation
6. Receive confirmation
```
